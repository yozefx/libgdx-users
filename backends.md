# Backends #

Backends provide access to high performance libraries like OpenGL etc.

Currently linGDX supports these backends:

  * [Android](backendAndroid.md)
  * [Angle](backendAngle.md)
  * [GWT](backendGwt.md)
  * [JOGL](backendJogl.md)
  * [LWJGL](backendLwjgl.md)
  * [Openal](backendOpenal.md)
